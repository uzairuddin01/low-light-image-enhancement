About Us.

- This project is aimed at creating a solution to tackling the problem of Low Light Photography Using Simplified Fully Convolutional Neural Networks

- The Neural Network model architecture is written in PyTorch and the application is deployed using Flask

- The model was trained on the [Sony_gt Dataset]

Our Process.

In this, we are doing low light image enhancement through multi Deep neural network layers, which extracts a parameter map with the same size as the given image. then apply this equation on the original image and extracted map using gamma functions.
